# Architecture Decision Records

This directory contains ADRs documenting significant architectural decisions for A Long Journey.

## Conventions

- Filename format: `adr-NNN-short-title.md` (zero-padded 3-digit number)
- Each ADR is immutable once accepted — if a decision is reversed, create a new ADR that supersedes the old one
- Keep the ADR table in `ARCHITECTURE.md` up to date

## Template

```markdown
# ADR-NNN: Title

**Date:** YYYY-MM-DD
**Status:** Proposed | Accepted | Superseded by ADR-XXX

## Context

What is the issue or decision that needs to be made?

## Decision

What was decided and why.

## Consequences

What are the tradeoffs? What becomes easier or harder as a result?
```
